# Gaurav Pawar — Portfolio

Minimal personal portfolio built with React, TypeScript, and Vite.

## Tech Stack

React 19 · TypeScript · Vite · CSS Custom Properties

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Output goes to `dist/`. Deploy to Vercel, Netlify, or any static host.
